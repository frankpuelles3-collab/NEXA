# NEXA
NEXA es inteligencia artificial personal creada para ayudar, enseñar y acompañar a personas de todas las edades. Responde preguntas, apoya emocionalmente y ofrece juegos educativos con puntaje. Incluye modo niño y modo adulto, perfiles con íconos personalizados y una interfaz inspirada en la tranquilidad de la playa. NEXA existe para el bien.!
